brainrender

view the code on Github

Brainrender is an easy-to-use python package for creating 3D rendering of neuroantomical data. It can be used to download and visualise data from several publicly available datasets with neuroanatomical data for different species. Brainrender can also be used to visualise user-generated content registered to a reference atlas.

scidraw.io

explore scidraw.io

SciDraw is a free repository of high quality drawings of animals, scientific setups, and anything that might be useful for scientific presentations, posters and publications. The main goal of scidraw.io is to facilitate the communication of scientific knowledge through high quality illustrations.

Head over to scidraw.io to explore over 200 hundred freely available drawings.
