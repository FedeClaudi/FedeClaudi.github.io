Brainrender

Brainrender is an easy-to-use python package for creating 3D rendering of neuroantomical data. It can be used to download and visualise data from several publicly available datasets with neuroanatomical data for different species. Brainrender can also be used to visualise user-generated content registered to a reference atlas.


For more up-to-date documentation and an extensive list of examples of how to use brainrender, please head to the github repository. You can also explore interactive scenes on this website.

Brainrender was developed to visualise data from the Allen Brain atlas and other datasets from the Allen institute. In addition brainrender can be used to visualise neuronal morphological data from neuromorph.org and from the mouselight project from the Janelia campus.


We are currently adding new features to brainrender to visualise neuroanatomical data for other species including: human, rat, zebra fish, drosophila and others.
